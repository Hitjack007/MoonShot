# MoonShot

Moonshot is a iOS app that is designed to help users see the history of the Apollo Missions at thier fingetips. Developed during Paul Hudsonâ€™s [100 Days of Swift](https://www.hackingwithswift.com/100) course, it showcases JSON Decoder integration with SwiftUI.

## Features
- Learn the History of the Apollo Missions
- Interact - Choose a mission, astronaut and more
- Clean, responsive UI

## Getting Started

1. Download or clone this repo.
2. Open the project folder in Xcode.
3. Press **Cmd + R** to build and run on the simulator or device.
